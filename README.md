## Description
DOSC（Deep Ocean Stereo Camera）database is a real-world deep-sea stereo image dataset designed for underwater image enhancement, color restoration, stereo matching, and depth-guided visual perception. The dataset contains stereo color images collected from real deep-ocean scenes, including seafloor environments, marine organisms, sediments, rocks, artificial structures, and underwater operation targets. DOSC captures typical deep-sea imaging degradations such as low illumination, non-uniform artificial lighting, wavelength-dependent color attenuation, red-channel loss, suspended particles, local overexposure, and depth-dependent visibility changes. It provides a realistic benchmark for developing and evaluating underwater vision algorithms in complex deep-sea environments.
